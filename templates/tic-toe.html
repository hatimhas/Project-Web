About

  • Designed and implemented a dynamic Tic Tac Toe game using HTML, CSS, and JavaScript, allowing for various game modes.
  • Developed the game grid by dynamically creating cells with event listeners for user interactions and updating the UI based on player actions.
  • Created JavaScript functions to handle player moves, update the game state, and determine winning combinations or a draw.
  • Integrated logic for switching players, tracking game status, and displaying messages for current player turns, game results, and mode selections.
  • Implemented an AI opponent with two difficulty levels: a basic random-move AI and a more advanced AI using the minimax algorithm to make optimal moves.
  • Added functionality to reset the game, clear the board, and allow players to select a new game mode or restart the game seamlessly.

  • HTML: Structuring the game board and interface elements to create a clear and organized layout.
  • CSS: Styling the Tic Tac Toe grid to ensure a visually appealing and consistent layout, including responsive design adjustments.
  • JavaScript: Handling user interactions, implementing game logic, and dynamically updating the game state and UI.
  • DOM Manipulation: Dynamically creating and managing game cells, and updating the game status based on user actions and game results.
  • Bootstrap: Applying responsive design principles to ensure the game layout adapts effectively across different screen sizes.
  • Algorithm: Implementing the Minimax algorithm for the AI in the Tic Tac Toe game to make optimal decisions based on game state and possible future moves.
